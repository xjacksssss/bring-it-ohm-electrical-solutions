# Bring It Ohm Electrical Solutions

A professional, production-ready Next.js website for Bring It Ohm Electrical Solutions, an electrical services company based in Walton on Thames, Surrey.

## Features

- **Modern Next.js 14+** with App Router
- **Bootstrap 5** for responsive design
- **Server and Client Components** for optimal performance
- **Contact Form** with email delivery via SMTP
- **SEO Optimized** with proper metadata and Open Graph tags
- **Responsive Design** - Mobile-first approach
- **Image Optimization** using Next.js Image component
- **Google Fonts** integration (Roboto Slab & Roboto)
- **Accessibility** features with ARIA labels and semantic HTML
- **Rate Limiting** on contact form to prevent spam
- **Form Validation** both client-side and server-side

## Services Offered

- Electrical installations and upgrades
- Emergency electrician services (24/7)
- EV charging point installations
- Lighting design and control systems
- Full and partial rewiring
- Rako smart home automation systems
- Electrical installation condition reporting (EICR)
- Inspection and testing
- Electrical issue rectification and certification

## Technology Stack

- **Framework:** Next.js 14+ (App Router)
- **Language:** JavaScript (ES6+)
- **Styling:** Bootstrap 5, Custom CSS
- **UI Components:** React-Bootstrap
- **Icons:** Heroicons
- **Email:** Nodemailer
- **Fonts:** Google Fonts (Roboto Slab, Roboto)
- **Image Optimization:** Next.js Image
